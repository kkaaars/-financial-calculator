# Финансовый Калькулятор

Лабораторная работа по созданию калькулятора больших чисел с фиксированной запятой.

Приложение работает в браузере и выполняет финансовые вычисления с высокой точностью (без ошибок стандартной плавающей точки).

## Как запустить

1. Скачайте файл `step_.html` из этого репозитория.
2. Откройте его двойным щелчком в любом браузере (Chrome, Firefox, Edge, Safari).

---

## Функционал

В работе реализованы все требования (Шаг 1, 2 и 3):

1.  **Точные вычисления:**
    *   Используется арифметика с фиксированной запятой (на базе `BigInt`).
    *   Точность внутренних расчетов — **10 знаков** после запятой.
    *   Поддержка чисел до ±1 триллиона.

2.  **Операции:**
    *   Сложение, вычитание, умножение, деление.
    *   Цепочка из 4-х чисел.
    *   **Приоритет операций:** Скобки `( ... )` и умножение/деление выполняются первыми.

3.  **Удобство:**
    *   Ввод чисел с пробелами (например, `1 000 000`).
    *   Защита от некорректного ввода (буквы, лишние символы).

4.  **Округление результата:**
    После расчета можно выбрать один из трех методов округления до целого:
    *   **Математическое** (обычное).
    *   **Бухгалтерское** (до ближайшего четного).
    *   **Усечение** (отбрасывание дробной части).
